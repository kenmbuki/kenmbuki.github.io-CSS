Book Summary

CSS completely changes the way elements are presented by a user agent.This can be executed ata basic level with the display property and in the different way by associating style sheets and inline styles. CSS has a number of importance :-

  • Improves structure of the page and increase the google rank
  • Brings about easier maintenance
  • Increases page accesibility and easier handling of multiple pages structure
  • Compact size - Thereby speed in downloading times and loading times in viewing
