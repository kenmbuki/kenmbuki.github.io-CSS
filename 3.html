    Inline Style specificity

    This is important. Totally DARK

    The default style declaration are the least influential. If author-rule applies to the anchors then this rule overrides the user defaults.

      To sum up, there are 5 levels to consider in terms of declaration weight

      1. Reader Important declarations
      2. Author important declarations
      3. Author normal declarations
      4. Reader Normal declarations
      5. User agent declarations

    Remember that in CSS2, inline style declarations have a specificity equal to ID selectors

    LVHA applies order sorting. One can also order as VLHA but the latter is preferred in CSS2 and also because the Mnemonic "LoVe--HA!" gained rather a wide currency
